# 3-Verliert

**3-Verliert** ist ein kleines Zwei-Spieler-Spiel, umgesetzt in **TypeScript** und **React**. 
Das Spiel wird auf einem 4x4-Spielfeld gespielt. Zwei Spieler („Kreuz“ und „Kreis“) setzen abwechselnd ihre Symbole. 
Ein Spieler verliert, sobald er drei seiner Symbole in einer Reihe, Spalte oder Diagonale direkt nebeneinander setzen muss. 
Das Spiel endet unentschieden, wenn alle Felder gesetzt sind, ohne dass ein Spieler verloren hat.

## Installation & Start

1. Repository klonen:

```bash
git clone https://github.com/kuerbi/3-verliert.git
```

2. Abhängigkeiten installieren:
```bash
npm install
```

3. Development Server starten:
```bash
npm run dev
```

4. Im Browser öffnen (normalerweise http://localhost:5173)

Viel Spaß beim Spielen.
